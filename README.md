# Модуль системы наведения для CubeSat 1U

Данный репозиторий содержит материалы командного кейса №1 «Модуль системы наведения» в рамках Московской предпрофессиональной олимпиады школьников (профиль «Инженерия», 2025/2026 уч. г.). Разработан программно-аппаратный комплекс – независимый модуль наведения лазерного целеуказателя для космического аппарата форм-фактора CubeSat 1U.

## Цель и задачи

### Цель работы
Разработать автономный модуль наведения лазерного луча в двух осях (наклон/поворот) с углом обзора ±40° по каждой оси, управляемый по радиоканалу, и интегрировать его в корпус CubeSat 1U.

### Задачи

1. **Механическое проектирование**: спроектировать механическую конструкцию, обеспечивающую крепление двух сервоприводов, лазера, контроллера и радиомодуля внутри корпуса 100×100×100 мм.

2. **Электроника**: разработать электрическую схему и собрать макет.

3. **Программное обеспечение**: написать ПО для RP2040-Zero (спутник) и Arduino Nano (наземная станция), реализующее алгоритмы сканирования (горизонтальное, вертикальное, диагональные) с шагом 10° и задержкой 3 секунды.

4. **Радиокоммуникация**: обеспечить приём команд и передачу телеметрии по радиоканалу (NRF24L01).

## Описание функций

Разработанное устройство выполняет следующие функции:

**Инициализация**: после подачи питания модуль устанавливает лазер в центр (0°, 0°), включает лазер и переходит в режим ожидания команд.

**Приём команд**: по радиоканалу (NRF24L01) принимаются пакеты, содержащие, код команды и параметры углов.

**Управление лазером**: по команде устанавливает заданные углы наклона и поворота с шагом 10°.

Режимы сканирования:

**Горизонтальный скан**: фиксированный вертикальный угол (0°), изменение горизонтального от –40° до +40° с шагом 10°.

**Вертикальный скан**: фиксированный горизонтальный угол (0°), изменение вертикального от –40° до +40° с шагом 10°.

**Диагональный скан 1**: одновременное изменение обоих углов от (-40°,-40°) до (40°,40°) с шагом 10°.

**Диагональный скан 2**: одновременное изменение углов от (-40°,40°) до (40°,-40°) с шагом 10°.

**Телеметрия**: после каждого шага сканирования или по запросу отправляются текущие углы и режим работы.

**Тестовые функции**: включение/выключение лазера, тест сервоприводов.



## Аппаратные средства

### Компоненты системы:

- **Arduino Nano** — наземная станция, принимающая команды START от ПК и передающая последовательности позиций по радио
- **Raspberry Pi Pico (RP2040)** — исполнительный модуль внутри CubeSat, управляющий двумя сервоприводами 9G Digital servo и лазером KY-008
- **Радиомодуль nRF24L01** — для односторонней связи от Arduino к RP2040
- **Сервоприводы 9G Digital servo** — привод для управления углами поворота и наклона
- **Лазер KY-008** — модуль лазерного целеуказателя
- **Собственный 3D-напечатанный корпус** — соответствует стандарту CubeSat 1U с учётом размещения лазера
